1o O que é HTML?

(Voltar)

HTML significa linguagem de marcação de hipertexto. É a linguagem de marcação padrão que descreve a estrutura de uma página web. O HTML consiste em uma série de elementos que informam ao navegador como exibir o conteúdo. Esses elementos HTML rotulam partes do conteúdo como “este é um título”, “este é um parágrafo”, “este é um link”, etc.

2o Estrutura Básica do HTML

(Voltar)

A estrutura básica do html é composta por diferentes elementos que formam o esqueleto de uma página da web. Abaixo está a estrutura básica do HTML, que inclui as principais seções de um documento HTML padrão:

  1. A <!DOCTYPE html> declaração define que este documento é um documento HTML5.
  2. O < html >< /html > é o elemento raiz de uma página HTML.
  3. O < head >< /head > é um elemento que contém meta informações sobre a página HTML.
  4. O < title >< /title > é um elemento que especifica um título para a página HTML ( é mostrado na barra de título do navegador ou na guia da página).
  5. O < body >< /body > elemento que define o corpo do documento e é um contêiner para todos os conteúdos visíveis, como títulos, parágrafos, imagens, hiperlinks, tabelas, listas, etc.
  6. O < h1 >< /h1 > elemento define um título grande.
  7. O < p >< /p > elemento define um parágrafo.

3o Cabeçalhos ou Títulos HTML

(Voltar)

Os títulos HTML são títulos ou legendas que você deseja exibir em uma página da web. Eles são definidos com as tags < h1 > até < h6 >. O < h1 > define o título mais importante. E o < h6 > define o título menos importante.

4o Atributos HTML

(Voltar)

Todos os elementos HTML podem ter atributos, eles fornecem informações adicionais sobre os elementos e são sempre especificados na tag inicial. Os atributos geralmente vêm em pares nome/valor como: nome="valor"

O atributo href

A < a > tag define um hiperlink. O href atributo especifica o URL da página para a qual o link leva:

Exemplo: < a href="https://www.w3schools.com">Visit W3Schools</a></p>.

O atributo src

A < img > tag é usada para incorporar uma imagem em uma página HTML. O src atributo especifica o caminho para a imagem a ser exibida:

Exemplo < img src="img_girl.jpg">

5o Elementos de formatação HTML

(Voltar)

Os elementos de formatação foram projetados para exibir tipos especiais de texto:

  1. < b > Texto em negrito
  2. < strong > Texto importante
  3. < i > Texto em itálico
  4. < em > Texto enfatizado
  5. < mark > Texto marcado
  6. < small > Texto menor
  7. < del > Texto excluído
  8. < ins > Texto inserido
  9. < sub > H2O Texto subscrito
  10. < sup > 22 sobrescrito
